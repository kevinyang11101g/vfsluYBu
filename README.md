# Python高校教室管理系统 python66

## 项目概述

本项目是基于Python开发的高校教室管理系统，采用B/S架构，后端使用Django框架，数据库为MySQL。系统致力于为高校提供便捷的教室资源管理，包括教师、学生、教室信息以及预约管理等功能。

## 技术栈

- Python
- MySQL
- B/S架构
- Django

## 功能模块

### 基础模块

- 首页
- 个人中心

### 管理模块

- 教师管理
- 学生管理
- 教室信息管理

### 申请管理

- 教师申请管理
- 学生申请管理

### 课时表管理

- 课时表查看与编辑

### 取消预约管理

- 教师取消预约管理
- 学生取消预约管理

## 系统角色

- 教师
- 学生
